# Claude Code 用户交互 - 快速参考卡

## 当 Claude Code 需要输入时

### 显示的消息格式

```
【Claude Code 需要你的输入】
Claude Code 需要你的输入。
请按 `问题ID=答案` 每行回复。

问题 1 (q1)
选择修改方式
可选项：
1. 直接修改 MODEL_IDS 列表
2. 添加 MODEL_DISPLAY_NAMES 字典
3. 修改 model_id_from_display_name 函数

请直接回复序号或选项文本。
```

### 用户操作

| 操作 | 说明 |
|------|------|
| 输入 `q1=1` | 选择选项 1 |
| 输入 `q1=2` | 选择选项 2 |
| 输入 `q1=3` | 选择选项 3 |
| 输入 `q1=自定义内容` | 输入自定义内容 |

### 程序处理

1. ✓ 验证格式
2. ✓ 将输入发送给 Claude Code
3. ✓ Claude Code 继续执行

## 当 Claude Code 需要批准时

### 显示的消息格式

```
【Claude Code 需要批准】
Claude Code 需要批准：
修改文件 main.py 中的 MODEL_DISPLAY_NAMES 字典

请回复 'yes' 或 'no'。
```

### 用户操作

| 操作 | 说明 |
|------|------|
| 输入 `yes` | 批准操作 |
| 输入 `no` | 拒绝操作 |

### 程序处理

1. ✓ 验证格式
2. ✓ 将批准结果发送给 Claude Code
3. ✓ Claude Code 继续执行

## 工作流程

```
Claude Code 返回请求
    ↓
程序显示请求消息
    ↓
用户在输入框中输入内容
    ↓
用户点击"发送"按钮
    ↓
程序验证格式
    ↓
程序将输入发送给 Claude Code
    ↓
Claude Code 继续执行
```

## 常见问题

### Q: 如果我输入错误的格式会怎样？

A: 程序会显示错误提示，例如：
```
❌ 输入错误：选项序号超出范围（1-3）。
```

### Q: 如果我超过 5 分钟没有输入会怎样？

A: 程序会显示超时提示：
```
⏱️ 输入超时，已取消
```

### Q: 我可以输入自定义内容吗？

A: 可以，如果问题支持"其他"选项，你可以输入自定义内容。

### Q: 我可以取消输入吗？

A: 目前不支持，但可以等待超时（5 分钟）。
