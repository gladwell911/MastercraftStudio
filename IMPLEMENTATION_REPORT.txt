================================================================================
1. 项目概述
================================================================================

本项目为新增的 Claude Code CLI 聊天功能实现了手机端远程控制功能，参考了现有
Codex CLI 的远程控制实现。用户可以通过手机或其他客户端通过 WebSocket 连接远程
控制 Claude Code 聊天。

================================================================================
2. 新增文件
================================================================================

    - 处理 Claude Code 远程协议的消息格式化和解析
    - 支持用户输入请求、批准请求的格式化和解析

2.2 claudecode_remote_client.py (8.3 KB)
    - 实现 Claude Code 远程控制的 WebSocket 服务器

    CLAUDECODE_REMOTE_CONTROL_HOST=0.0.0.0 (默认)

================================================================================
6. 使用示例
================================================================================

6.1 启动应用
    set CLAUDECODE_REMOTE_CONTROL_TOKEN=your_secret_token
    python main.py

6.2 连接客户端
    ws://localhost:18080/ws?token=your_secret_token

6.3 发送命令
    {
      "action": "message",
      "text": "你好，请帮我分析这个代码"
    }
