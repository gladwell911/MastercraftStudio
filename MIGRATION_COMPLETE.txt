================================================================================
                        快速开始指南
================================================================================

第一步：设置环境变量

Windows PowerShell:
  $env:REMOTE_CONTROL_TOKEN="your_secret_token"
  $env:REMOTE_CONTROL_DOMAIN="rc.tingyou.cc"

Windows CMD:
  set REMOTE_CONTROL_TOKEN=your_secret_token
  set REMOTE_CONTROL_DOMAIN=rc.tingyou.cc

Linux/Mac:
  export REMOTE_CONTROL_TOKEN="your_secret_token"
  export REMOTE_CONTROL_DOMAIN="rc.tingyou.cc"

第二步：启动程序
  python main.py

第三步：获取远程控制地址
  - 在程序菜单中点击"复制远程控制地址"
  - 在手机端程序中粘贴该地址

================================================================================
                        新方案优势
================================================================================

对比 Cloudflared 方案：

✅ 无需额外进程
   - 不需要运行 cloudflared.exe
   - 资源占用更低

✅ 配置更简单
   - 只需设置两个环境变量
   - 支持多种域名格式

✅ 连接更快
   - 直接连接，无中间转发
   - 延迟更低

✅ 维护成本低
   - 无需管理 cloudflared 进程
   - 无需处理 token 文件

✅ 灵活性高
   - 支持自定义端口
   - 支持多种协议（HTTP/HTTPS/WS/WSS）

================================================================================
                        验证配置
================================================================================

检查环境变量是否正确设置：

PowerShell:
  echo $env:REMOTE_CONTROL_TOKEN
  echo $env:REMOTE_CONTROL_DOMAIN

CMD:
  echo %REMOTE_CONTROL_TOKEN%
  echo %REMOTE_CONTROL_DOMAIN%

检查域名是否可访问：
  ping tingyou.cc

检查端口是否开放：
  Test-NetConnection -ComputerName tingyou.cc -Port 18080

================================================================================
                        故障排除
================================================================================

问题：连接失败
解决：
  1. 检查环境变量是否正确设置
  2. 检查域名是否可访问（ping tingyou.cc）
  3. 检查防火墙是否阻止 18080 端口
  4. 查看程序状态栏的错误信息

问题：令牌错误
解决：
  1. 确保 REMOTE_CONTROL_TOKEN 不为空
  2. 手机端使用的令牌必须与服务端一致

问题：端口被占用
解决：
  1. 设置 REMOTE_CONTROL_PORT 环境变量为其他端口
  2. 或关闭占用该端口的其他程序

================================================================================
                        安全建议
================================================================================

1. 使用强令牌
   - 建议使用至少 32 个字符的随机字符串
   - 可以使用 UUID 生成器生成

2. 定期更换令牌
   - 定期更新 REMOTE_CONTROL_TOKEN 值

3. 使用加密连接
   - 在公网环境中，建议使用 WSS（加密）连接
   - 设置 REMOTE_CONTROL_DOMAIN=https://rc.tingyou.cc

4. 防火墙配置
   - 只在需要时开放 18080 端口
   - 考虑使用 VPN 或其他安全隧道
